# Basaa documentation — downloaded articles

Scientific articles on Basaa (A43) gathered for the treebank project. The starting bibliography is
in `resources.md`. Files actually present:

| File | Reference | Why it matters for a treebank |
|---|---|---|
| `basaa.pdf` | **Makasso & Lee 2015**, "Basaá", *J. of the IPA* 45(1): 71–79 | **The seed text.** Contains the fully glossed, translated "North Wind and the Sun" (Basaa IPA / morpheme gloss / English) + a word list. Cleanest interlinear data we have. Audio is members-only. |
| `Hamlaoui_Makasso_2014_HighTone_PhonPhrases_Basaa.pdf` | Hamlaoui & Makasso 2014, "High tone spreading and Phonological Phrases in Bàsàá" (TAL 2014) | Clean glossed SVO examples (e.g. *sóGól à-ǹ-tÉhÉ á-ÓÓNgÉ* 'the grandfather saw the children'); class/agreement/CONN morphology; prosodic phrasing = constituency cues. |
| `BULBasaa_corpus_Hamlaoui_et_al_2018_LREC.pdf` | Hamlaoui et al. 2018, BULBasaa corpus (LREC) | Describes the ~50 h Basaa–French speech corpus (ELRA). Roadmap for the audio+transcription layer; basis for an ELRA request. |
| `Adda-etal_2022_Tone_prediction_orthographic_conversion_Basaa.pdf` | "Tone prediction and orthographic conversion for Basaa" (arXiv 2210.06986) | NLP-side: minimal-tone ↔ full-tone orthography conversion. Directly relevant to normalising treebank text and to tone restoration. |
| `Hamlaoui_2020_realisation_r_francais_basaaphones.pdf` | Hamlaoui 2020, /r/ in the French of Basaa speakers (SHS Web of Conf.) | Phonetics of Basaa-accented French; peripheral, but documents the speaker community and the semi-spontaneous data. |
| `aflin_2033-8732_1986_num_10_1_928.pdf` | Africana Linguistica X (1986), Persée | Historical-comparative Bantu A-zone classification context for Basaa (A43). Background. |

## High-value sources still to obtain

- **Hamlaoui & Makasso 2015**, "Focus marking and the unavailability of inversion structures in
  Bàsàá (A43)," *Lingua* 154: 35–64 — the central syntactic analysis (the `basaa` morphology module
  in BULB tooling implements it). Paywalled on ScienceDirect; an author PDF is linked from
  <https://sites.google.com/site/fhaml044/publications>. **Get this** — it governs how fronted/focus
  constituents should attach in the treebank.
- **"Pronominal F-markers in Basaá"** (Hamlaoui & Makasso) — heavily glossed, with an explicit
  gloss-abbreviation key (FUT2, GEN, INF, LOC, OBL, PASS, PRS, class numerals). ResearchGate copy.
- **Hyman 2003**, "Basaá (A43)," in *The Bantu Languages* (Routledge) — baseline morphology reference.
